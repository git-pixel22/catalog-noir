# UI Rehaul Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add pixel art character sprites, Ace Attorney–style dialogue UI, ambient sound, scene transitions, and a consistent blue/noir palette across all 5 game scenes.

**Architecture:** All changes are in `frontend/index.html` (single 1244-line file). Characters are drawn programmatically into Phaser CanvasTextures using compact rect-list pixel data — no image files required. PNG override is supported by dropping files in `frontend/assets/`. The existing game logic, backend, and SFX engine are untouched.

**Tech Stack:** Phaser 3.87 (CanvasTexture, Sprite, camera fade), Web Audio API (ambient loop), vanilla JS — no new dependencies.

```js
    ambientLoop() {
      const c = get();
      let active = true;

      const master = c.createGain();
      master.gain.setValueAtTime(0.12, c.currentTime);

      const lp = c.createBiquadFilter();
      lp.type = 'lowpass';
      lp.frequency.setValueAtTime(820, c.currentTime);
      lp.connect(master);
      master.connect(c.destination);

      function note(freq, dur, vol) {
        if (!active) return;
        const o = c.createOscillator();
        const g = c.createGain();
        o.type = 'sine';
        o.frequency.setValueAtTime(freq, c.currentTime);
        g.gain.setValueAtTime(vol, c.currentTime);
        g.gain.exponentialRampToValueAtTime(0.001, c.currentTime + dur);
        o.connect(g); g.connect(lp);
        o.start(); o.stop(c.currentTime + dur + 0.05);
      }

      // Bass pulse every 2s
      note(55, 0.7, 0.5);
      const bassTimer = setInterval(() => note(55, 0.7, 0.5), 2000);

      // Sparse pentatonic melody — A minor: A2 C3 D3 E3 G3
      const melody = [110, 130, 147, 165, 196];
      function scheduleMelody() {
        if (!active) return;
        note(melody[Math.floor(Math.random() * melody.length)], 1.1, 0.18);
        setTimeout(scheduleMelody, 1400 + Math.random() * 1800);
      }
      setTimeout(scheduleMelody, 600);

      return {
        stop() {
          active = false;
          clearInterval(bassTimer);
          master.gain.setValueAtTime(0, c.currentTime);
        },
      };
    },
```
